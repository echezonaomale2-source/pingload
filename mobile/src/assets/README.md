# Mobile assets

| File | Purpose |
|------|---------|
| `logo.png` | Circular P logo mark (transparent, square) |
| `splash.png` | Native splash — logo mark centered on white |
| `icon.png` | Expo / iOS app icon (1024×1024) |
| `adaptive-icon.png` | Android adaptive icon foreground |
| `brandAssets.js` | Shared logo imports and sizing helpers |

Regenerate logo assets from the source splash image:

```bash
python mobile/scripts/extract-logo-mark.py
```

Brand tagline: **Fast. Secure. Reliable.**
